¿Sobrevivirías al viaje del Titanic?

Introduce los parámetros

Introduce los datos en el formulario. El modelo utilizado es Arbol de decisión, en el cual, se basa en los parámetros introducidos para determinar si sobrevivirías

Las instrucciones para rellenar los campos son las siguientes:

  • Clase de billete (1 primera clase, 2 segunda clase, 3 tercera clase)
  • Edad (Edad en años)
  • Esposos a bordo
  • Hijos a bordo
  • Precio billete $
  • Hombre (1 si, 0 no )
  • Mujer (1 si, 0 no )
  • puerto_salida_C (1 si, 0 no )
  • puerto_salida_Q (1 si, 0 no )
  • puerto_salida_S (1 si, 0 no )

Aquellos campos en los que se introduzca 0 o 1, en el caso de ser mayor que 1, será traducido como 1. En otro caso obtendremos como resultado error.
