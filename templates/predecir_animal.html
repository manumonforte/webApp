Clasificar Animal

Introduce los parámetros

Introduce los datos para predecir de que tipo de animal se trata. El modelo utilizado es Arbol de decisión, en el cual, se basa en los parámetros introducidos para determinar que clase de animal es.

Las instrucciones para rellenar los campos son las siguientes:

  • Pelo (1 tiene, 0 no tiene)
  • Plumas (1 tiene, 0 no tiene)
  • Huevos (1 tiene, 0 no tiene)
  • Leche (1 tiene, 0 no tiene)
  • Vuela (1 tiene, 0 no tiene)
  • Acuático (1 es acuático, 0 no es acuático)
  • Depredador (1 es depredador, 0 no es depredador)
  • Dientes (1 tiene, 0 no tiene)
  • Columna Vertebral (1 tiene, 0 no tiene)
  • Pulmones (1 Respiración pulmonar, 0 no respiración pulmonar)
  • Venenoso (1 es venenoso, 0 no es venenoso)
  • Aletas (1 tiene, 0 no tiene)
  • Patas (Numero de patas)
  • Cola (1 tiene, 0 no tiene)
  • Doméstico (1 tiene, 0 no tiene)

Aquellos campos en los que se introduzca 0 o 1, en el caso de ser mayor que 1, será traducido como 1. En otro caso obtendremos como resultado error.
